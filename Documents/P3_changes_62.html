Our client is Jakob Erikstad, working chairman of the board of Edyou AS.

Javacsript

Erlend hadde hovedansvaret for Javascript. Her ble stort sett det meste implementert som planlagt, men dessverre rakk vi ikke fullføre alt. Spesielt gjelder dette FAQ funksjonaliteten som ekspanderer/kollapser tekstbokser og popup vinduet. form submission, scroll back to top og navbar funksjonalitet ble implementert.

Når det gjelder innhold og css ble heller ikke det fullført grunnet tidsbegrensing på FAQ-siden. Dette var Marius sitt ansvar.

Modularitet

Modularitet ble løst gjennom å bruke javascript og HTMLElement. Feks navbar og footer som brukes på alle sider ble lagd som egne komponenter slik at de kunne gjenbrukes på alle sider.

Changes

Det var svært få endringer fra planen. Den største var kanskje at vi innså at footer og navbar kunne lages som komponenter og dette ble gjort i stedet for å skrive de inn hver gang.
